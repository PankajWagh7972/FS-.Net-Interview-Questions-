# 1️⃣ What is .NET Core?

**Answer:**

.NET Core is a **cross-platform, open-source, high-performance framework** for building:

* Web APIs
* Microservices
* Cloud applications
* Console apps

Runs on:

* Windows
* Linux

# 5️⃣ Dependency Injection (DI)

### What is DI?

Injecting dependent objects rather than creating manually.

### Example:

```csharp
builder.Services.AddScoped<IUserService, UserService>();
```

```csharp
public class UserController : ControllerBase
{
    private readonly IUserService _service;

    public UserController(IUserService service)
    {
        _service = service;
    }
}
```

### DI Lifetimes:

| Lifetime  | Meaning                     |
| --------- | --------------------------- |
| Singleton | One instance for entire app |
| Scoped    | One per request             |
| Transient | New instance every time     |

# 6️⃣ Middleware Pipeline

### What is Middleware?

Components that process requests & responses.

```csharp
app.Use(async (context, next) =>
{
    Console.WriteLine("Before Request");
    await next();
    Console.WriteLine("After Request");
});
```

# 12️⃣ Global Exception Handling Middleware

```csharp
app.UseExceptionHandler(error =>
{
    error.Run(async context =>
    {
        context.Response.StatusCode = 500;
        await context.Response.WriteAsync("Something went wrong");
    });
});
```

### 2. Global Exception Middleware

```csharp
public class ExceptionMiddleware
{
    public async Task Invoke(HttpContext context)
    {
        try { await _next(context); }
        catch(Exception ex)
        {
            context.Response.StatusCode = 500;
            await context.Response.WriteAsync(ex.Message);
        }
    }
}
```
